# 🚗 DriveX — Car Service Landing

Современный адаптивный лендинг для автосервиса с акцентом на конверсию и удобство пользователей.

## ✨ Возможности

- 📱 Полностью адаптивный дизайн
- ⚡ Плавная анимация элементов
- ❓ FAQ с аккордеоном
- 📩 Форма записи
- 🤖 Отправка заявок в Telegram
- 📈 Индикатор прокрутки страницы
- 🎨 Современный UI в темной цветовой гамме
- 🚀 Развернут на GitHub Pages

## 🛠️ Технологии

- HTML5
- CSS3
- JavaScript (ES6)
- Telegram Bot API
- Git & GitHub Pages
